# Configuration

NICTO system configuration files.

## Key Files

| File | Purpose |
|------|---------|
| `pyproject.toml` | Python project metadata, dependencies, build config |
| `.gitignore` | Git exclusion rules |
| `docker-compose.yml` | Docker multi-service setup |
| `Dockerfile` | Container build instructions |
| `.pre-commit-config.yaml` | Pre-commit hooks |
| `package.json` | Node.js dependencies (desktop app) |
| `hardware_config.json` | Auto-detected GPU/CPU config for training |
