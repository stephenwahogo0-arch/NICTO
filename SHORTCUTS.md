# KYROS Keyboard Shortcuts

## Global Shortcuts (Desktop Mode)
| Shortcut | Action |
|----------|--------|
| `Space` (double-tap) | Activate KYROS headless mode |
| `Ctrl+Q` | Quit KYROS |
| `Ctrl+H` | Toggle headless mode on/off |
| `Ctrl+Shift+V` | Toggle voice input on/off |
| `Ctrl+N` | New chat session |

## Web UI Shortcuts
| Shortcut | Action |
|----------|--------|
| `Enter` | Send message |
| `Shift+Enter` | New line in message |
| `Ctrl+N` | New chat (reload page) |
| `Ctrl+Shift+V` | Toggle voice input (browser speech) |
| `Space` (double-tap) | Activate KYROS if headless mode is running |

## Terminal Mode Shortcuts
| Shortcut | Action |
|----------|--------|
| `Ctrl+C` | Interrupt current operation |
| `Ctrl+D` | Exit KYROS |
| `Up/Down arrows` | Scroll through command history |

## Voice Chat Shortcuts
| Shortcut | Action |
|----------|--------|
| `Hold Space` (while speaking) | Voice-to-voice chat: talk to KYROS |
| `Release Space` | KYROS processes and replies with voice |

## Headless Mode (Desktop Avatar)
The headless avatar runs in its own window and responds to:
- **Space double-tap** — activates from any app
- **Voice input** — speak naturally, KYROS responds via TTS
- **Text input** — type messages in the companion web UI
